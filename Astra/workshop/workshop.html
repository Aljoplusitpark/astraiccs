Introduction to Flutter

Flutter is a modern, open-source UI software development kit created by Google. It allows developers to build beautiful, native-looking apps for iOS, Android, web, Windows, macOS, and Linux from a single codebase. Join the workshop conducted by Plus IT Park, Thrissur on Astra ‘23

This workshop is suitable for:

  • Beginners with no prior programming experience
  • Developers who are interested in learning Flutter
  • Developers who want to learn how to build cross-platform apps

This workshop is suitable for:

Workshop On UI/UX

This workshop will introduce you to the basics of user interface (UI) and user experience (UX) design. UI design is the process of creating the visual and interactive elements of a product or service. UX design is the process of designing products and services that are easy to use and enjoyable for people.

This workshop is suitable for:

  • Students who are interested in learning more about UI/UX design
  • Professionals who want to learn how to apply UI/UX design principles to their work

Workshop on Total Station

A total station is a surveying instrument that combines the functions of a theodolite and an electronic distance meter (EDM) into a single unit. It is used to measure horizontal and vertical angles, distances, and elevations.

This workshop is suitable for:

  • Beginners with no prior experience with total stations
  • Students who are interested in learning more about total station surveying
  • Professionals who want to learn how to use total stations for their work
